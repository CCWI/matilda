# MATILDA-Korpus

Korpus-Handler for Machine-learning Advisor To Identify Library-related Decision Alternatives.

## Getting started
Deploy service on platform:
1. Prepare Environment
    1.1. Install Java 11
    1.2. Maven (minimum version 3.5.4)
    1.3. Install and start MongoDB
2. Check out MATILDA-repository
4. Build via Maven (mvn install)
5. Start Service (mvn spring-boot:run)
6. Navigate e.g. to Swagger-Endpoint (ENDPOINT: /swagger-ui.html)
